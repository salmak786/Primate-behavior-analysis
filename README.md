# Primate-behavior-analysis
Comparing Statistical Learning Models for the Classification of Primate Behavior

## Overview

This project investigates the use of statistical learning methods to classify behavioral states in Moor macaques (Macaca maura) using ecological and demographic predictors.

## Research Questions

1. How accurately can statistical learning models predict behavioral categories?
2. How does human disturbance influence behavioral classification?
3. Which variables contribute most to model predictions?

## Methods

- Random Forest
- Neural Network
- Deep Neural Network
- Lasso, Ridge and Elastic net
- Multinomial Logistic Regression
